# ISA projekt

### Libpcap

#### Debian, Ubuntu
```
$ apt-get install libpcap-dev
```

#### Fedora, Centos
```
$ dnf install libpcap-devel
```

#### Linking
```
$ g++ source_files.cpp -lpcap
```
